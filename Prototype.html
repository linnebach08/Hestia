Prototype

  • We will develop a heart rate monitor embedded system. It might not be in the form of a wearable bracelet at the time of prototype completion but it will be able to accurately measure heart rate.
  • We will create our database server and put simulated data into it.
  • We will develop a basic application which can display simulated data stored in our database.
  • We will develop a plan for how to make the water intake sensor and possibly design a schematic for it.
