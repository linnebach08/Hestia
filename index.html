About Hestia

Hestia is a household management application developed as a capstone project at the University of Utah. This application helps you manage many aspects of your household and includes budget, task, calendar, grocery list, and meal planning features. You can use this application on web and mobile for both Android and IOS.

Backend

The Backend of the Hestia application was designed by Shawn Michael Linnebach III and Bridger Peterson. It constists of a Microsoft SQL database and a RESTful Web API both hosted by Microsoft Azure. The database consists of over 25 tables that store all the information about a users household, budgets, schedule, to-dos grocery lists, and meals so that this information can be correctly retrieved every time a user logs in. Then, the Web API provides over 60 endpoints that the frontend can use to create, retreive, modify, and delete the data stored in the database.

Web Frontend

The Frontend Web Portal of the Hestia application was designed by Liam Zhou and Pierce Bringhurst. It constists of an Identity Server for User login and a web application. The Identity Server handles user authentication and password encryption to ensure that user data is secured. Then the web application is designed using angular and allows a user to login and scheduled events, create to-dos, create recipes, schedule meals, create a grocery list, and create budgets. Mobile Frontend

The Mobile Hestia application was designed by Thang Nguyen. It constists of a mobile application for both Android and IOS. This was designed using NativeScript which allows the same code to be used for the application on both mobile platforms. The mobile apps allow the user to login and scheduled events, create to-dos, and create budgets. The mobile downloads can be found in this Google Drive folder. app-debug.apk is the Android download and homelymobile.zip is the IOS download.
